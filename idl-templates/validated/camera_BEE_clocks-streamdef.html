Stream camera.BEE.clocks

Description

This subsystem comprises 25 computers (one per raft, not including spares), each hosting a fiber interface card to receive the image data generated by a raft. These computers would buffer the image data in memory and transfer the data into the Data Management stream. Iitial data quality assessment), may also be carried out on these hosts. The state of the clocks are monitored in this subsystem.

Update frequency

This telemetry stream publishes a new record every 10 seconds

Publishers

Only one instance of this stream is published.
