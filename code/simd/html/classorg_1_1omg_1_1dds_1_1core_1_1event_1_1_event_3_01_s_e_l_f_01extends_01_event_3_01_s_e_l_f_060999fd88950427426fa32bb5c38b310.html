Detailed Description

Event is the abstract root class for all communication event objects. All concrete kinds of Eventclasses extend this class.

Each concrete org.omg.dds.core.Entity is associated with a set of Status objects whose value represents the "communication status" of that entity. These status values can be accessed with corresponding methods on the Entity. The changes on these status values are the ones that both cause activation of the corresponding org.omg.dds.core.StatusCondition objects and trigger invocation of the proper Listener objects to asynchronously inform the application. The changes are notified via concrete subclasses of the Event type.
