Detailed Description

This interface is the base for Topic, ContentFilteredTopic, and MultiTopic.

TopicDescription represents the fact that both publications and subscriptions are tied to a single data type. Its attribute typeName defines a unique resulting type for the publication or the subscription and therefore creates an implicit association with a TypeSupport. TopicDescription has also a name that allows it to be retrieved locally.

Parameters:
<TYPE>The concrete type of the data that will be published and/ or subscribed by the readers and writers that use this topic description.

void org::omg::dds::topic::TopicDescription< TYPE >::close ( )

Dispose the resources held by this object.

A TopicDescription cannot be closed if it is in use by any DataWriters or DataReaders. With respect to Topics specifically: a Topic cannot be closed if it has any remaining ContentFilteredTopics or MultiTopics related to it.
