Detailed Description

A Condition is a root interface for all the conditions that may be attached to a WaitSet. This basic interface is specialized in the following interfaces that are known by the middleware: GuardCondition, StatusCondition, and ReadCondition.

A Condition has a triggerValue that can be true or false and is set automatically by the Service.
