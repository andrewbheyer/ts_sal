Detailed Description

[optional] Specifies the value of the "strength" used to arbitrate among multiple org.omg.dds.pub.DataWriter objects that attempt to modify the same instance of a data object (identified by org.omg.dds.topic.Topic + key). This policy only applies if the Ownership#getKind() is Ownership.Kind#EXCLUSIVE. The default value of the ownership strength is zero.
