template<typename Array >
boost::python::object detail::make_array_aux ( Array  array)

Create a callable Boost.Python object that will return an array_proxy type when called.

Note
This function will conditionally register array_proxy types for conversion within Boost.Python. The array_proxy will extend the life of the object from which it was called. For example, if foo is an object, and vars is an array, then the object returned from foo.vars will extend the life of foo.
