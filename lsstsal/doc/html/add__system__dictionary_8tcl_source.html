21 <H2>Introduction</H3>
22 <P>
23 This document describes the keywords used to refer to items in the
24 Service Abstraction Layer (SAL) mediated services. e.g. Subsystem Telemetry.
25 <P>
26 The approved names for all subsystems, components, devices, actions,
27 are defined in this dictionary. SAL generated code is validated against
28 the dictionary to ensure system wide consistency.
29 <P>
30 The dictionary will evolve as the individual subsystem descriptions
31 are completed to increasing levels of detail.

543 proc add_system_dictionary_appendix { fout } {
544 global TSYSDIC SYSDIC
545  foreach section "subsystems devices properties actions units" {
546  puts $fout $TSYSDIC($section)
547  puts $fout "<P><UL>"
548  foreach i [split $SYSDIC($section) \n] {
549  puts $fout "<B>[lindex $i 0]</B>[lrange $i 1 end]<BR>"
550  }
551  puts $fout "</UL><P>"
552  }
553 }
